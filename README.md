# Universal Android Debloater 

Bash script using ADB for unrooted smartphones. 

## Supported brand : 
- Samsung
- Xiaomi
- Huawei

## Specific debloat lists : 
- Facebook related
- T-Mobile related
- Microsoft related
- Google related
- Miscellaneous
